Introduction

This report analyzes IPEDS enrollment data to explore how campus diversity changed before and after the Supreme Court’s 2023 decision in Students for Fair Admissions v. Harvard (SFFA). The goal is not to make strong causal claims, but rather to use descriptive analysis to see whether 2024 enrollment patterns look materially different from prior years and whether those changes are unusually large relative to benchmark institutions.

I begin by acquiring and preparing the IPEDS data, then create two institutional indicators required for the assignment: one for CUNY institutions and another for California public institutions. After that, I perform an initial exploratory data analysis and answer a set of descriptive questions using inline values and formatted tables. I conclude with an Op-Ed written from the perspective of a college president.

Data Acquisition

Task 1: Acquire the IPEDS data

Show code 
#' Acquire IPEDS Data for MP#01
#'
#' This function will acquire and standardize all data for MP#01
#' from IPEDS (https://nces.ed.gov/ipeds/datacenter/DataFiles.aspx)
#'
#' We're starting in 2010 as the data seems to be reasonably complete
#' after that point.
acquire_ipeds_data <- function(start_year = 2010, end_year = 2024) {
  data_dir <- file.path("data", "mp01")

  if (!dir.exists(data_dir)) {
    dir.create(data_dir, showWarnings = FALSE, recursive = TRUE)
  }

  YEARS <- seq(start_year, end_year)

  EFA_ALL <- map(YEARS, function(yy) {
    if (yy <= 2022) {
      ef_url <- glue("https://nces.ed.gov/ipeds/datacenter/data/EF{yy}A.zip")
    } else {
      ef_url <- glue(
        "https://nces.ed.gov/ipeds/data-generator?year={yy}&tableName=EF{yy}A&HasRV=0&type=csv"
      )
    }

    ef_file <- file.path(data_dir, glue("ef{yy}a.csv.zip"))

    if (!file.exists(ef_file)) {
      message(glue("Downloading Enrollment Data for {yy} from {ef_url}"))
      download.file(ef_url, destfile = ef_file, quiet = TRUE, mode = "wb")
    }

    read_csv(ef_file, show_col_types = FALSE) |>
      mutate(
        year = yy,
        enrollment_m_aian = EFAIANM,
        enrollment_f_aian = EFAIANW,
        enrollment_m_asia = EFASIAM,
        enrollment_f_asia = EFASIAW,
        enrollment_m_bkaa = EFBKAAM,
        enrollment_f_bkaa = EFBKAAW,
        enrollment_m_hisp = EFHISPM,
        enrollment_f_hisp = EFHISPW,
        enrollment_m_nhpi = EFNHPIM,
        enrollment_f_nhpi = EFNHPIW,
        enrollment_m_whit = EFWHITM,
        enrollment_f_whit = EFWHITW,
        enrollment_m_2mor = EF2MORM,
        enrollment_f_2mor = EF2MORW,
        enrollment_m_unkn = EFUNKNM,
        enrollment_f_unkn = EFUNKNW,
        enrollment_m_nral = EFNRALM,
        enrollment_f_nral = EFNRALW
      ) |>
      filter(
        (EFALEVEL %in% c(2, 12)) | (LINE %in% c(1, 15))
      ) |>
      mutate(
        level = case_when(
          EFALEVEL == 2 ~ "all undergraduate",
          EFALEVEL == 12 ~ "all graduate",
          LINE %in% c(1, 15) ~ "first year undergrad"
        )
      ) |>
      select(
        institution_id = UNITID,
        year,
        level,
        starts_with("enrollment_")
      ) |>
      group_by(institution_id, year, level) |>
      summarize(across(starts_with("enrollment_"), sum), .groups = "drop")
  }) |>
    bind_rows()

  DESC_ALL <- map(YEARS, function(yy) {
    if (yy <= 2022) {
      hd_url <- glue("https://nces.ed.gov/ipeds/datacenter/data/HD{yy}.zip")
    } else {
      hd_url <- glue(
        "https://nces.ed.gov/ipeds/data-generator?year={yy}&tableName=HD{yy}&HasRV=0&type=csv"
      )
    }

    hd_file <- file.path(data_dir, glue("hd{yy}.csv.zip"))

    if (!file.exists(hd_file)) {
      message(glue("Downloading Institutional Descriptions for {yy} from {hd_url}"))
      download.file(hd_url, destfile = hd_file, quiet = TRUE, mode = "wb")
    }

    suppressWarnings(
      read_csv(
        hd_file,
        show_col_types = FALSE,
        locale = locale(encoding = if_else(yy == 2024, "utf-8", "windows-1252"))
      ) |>
        mutate(year = yy, INSTNM = INSTNM) |>
        select(
          institution_id = UNITID,
          institution_name = INSTNM,
          state = STABBR,
          year
        )
    )
  }) |>
    bind_rows()

  inner_join(
    EFA_ALL,
    DESC_ALL,
    join_by(institution_id == institution_id, year == year)
  )
}

IPEDS <- acquire_ipeds_data()

The code above downloads and prepares the IPEDS enrollment and institution description files and stores them in a local data/mp01 folder so that future renders do not repeatedly hit the IPEDS servers.

Task 7: Op-Ed

An Op-Ed from the President of Baruch College

As President of CUNY Bernard M Baruch College, I believe discussions about diversity should start with clear data. For this analysis, I define diversity as the share of undergraduate students who are neither White nor Asian American using IPEDS racial and ethnic categories. That definition is not perfect, but it provides a transparent way to track changes over time and compare institutions using consistent data.

In 2024, Baruch enrolled 16,496 undergraduate students. Of those students, 18.3% were classified as White, 50.1% were women, and 40.3% fell into my working definition of diversity. These numbers reflect the role Baruch plays as a large public institution serving students from across New York City.

Many observers have asked whether the SFFA decision produced an immediate shift in enrollment. At Baruch, our first-year class changed from 2,614 students in 2023 to 2,532 in 2024. The share of first-year students in my diversity measure moved from 40.9% to 40.9%, while the White share moved from 16.1% to 17.9%. These relatively small changes suggest that any immediate effect at Baruch was modest.

Long-term trends also matter. In 2010, the share of Baruch undergraduates captured by this diversity measure was 25.5%. By 2024 it had risen to 40.3%, while the White share fell from 29.8% to 18.3%. When compared with institutions such as UC Berkeley and San Diego State, which have operated without affirmative action for decades, these results suggest that year-to-year changes are often gradual and influenced by many factors beyond a single policy decision.

This mini-project shows how a fairly simple descriptive workflow can still generate useful institutional insight. By combining enrollment data, institutional identifiers, and clear summary measures, I was able to answer targeted questions about Baruch, CUNY, California public universities, and national state-level patterns.

The main substantive takeaway is that 2024 enrollment shifts should be interpreted cautiously. Some institutions may show changes after SFFA, but meaningful interpretation requires comparison to longer-term trends and to benchmark institutions that were already operating under different admissions constraints. That broader context is what turns raw numbers into analysis.
